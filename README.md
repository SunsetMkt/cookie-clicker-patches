# cookie-clicker-patches

Educational patches to make Cookie Clicker Steam version run in browser.

These patches only enable features that are compatible with web browsers. It won't enable UIs that won't work.

## Steps

### Buy the game

<https://store.steampowered.com/app/1454400/Cookie_Clicker/>

### Copy

Copy `Cookie Clicker\resources\app\src\*` and `Cookie Clicker\resources\app\steam\*` to `public`.

### Add files

Copy `patches\put-all-to-public\*` to `public`.

### Apply patches

Apply `patches\apply-these-patches\*.patch`.

### Optional: Beta

Copy `index.html` to `beta.html` to enable beta mode.

### Host

Host `public` on a web server.
